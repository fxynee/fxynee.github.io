hexo的安装与使用

安装hexo

  • hexo是一个node.js的包
  • 包安装方法：
    • node.js中的包管理器：npm（node package manager）
    • npm install包名（参数）
  • hexo
    • 包名：hexo-cli
    • 输入命令：npm install hexo-cli -g
    • 进入指定的盘符
      • 进入D盘则输入d:
      • cd .当前文件夹
      • cd ..返回上一级
      • cd \返回根目录
      • cd+空格+文件夹名称

使用hexo

  • 输入命令：hexo init，出现“Start blogging with Hexo!”字样即为成功
  • 输入命令：hexo generate生成静态网页
  • 输入命令：hexo server创建web服务以预览
